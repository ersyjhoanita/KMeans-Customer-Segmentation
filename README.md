# KMeans-Customer-Segmentation
Project ini merupakan project latihan pada mata kuliah Machine Learning yang berfokus pada penerapan algoritma K-Means untuk melakukan segmentasi pelanggan.

Tujuan utama dari project ini adalah mempelajari bagaimana algoritma unsupervised learning bekerja dalam mengelompokkan data berdasarkan kemiripan fitur. Dengan melakukan segmentasi pelanggan, perusahaan dapat memahami pola perilaku konsumen, sehingga strategi pemasaran dapat dibuat lebih tepat sasaran dan sesuai dengan kebutuhan tiap kelompok pelanggan.

Pada project ini, saya menggunakan beberapa tahapan dasar machine learning, yaitu:
1. Exploratory Data Analysis (EDA) — melihat karakteristik data pelanggan
2. Data preprocessing — scaling, cleaning, dan pemilihan fitur yang relevan
3. Menentukan jumlah cluster optimal menggunakan:
   - Elbow Method
   - Silhouette Score
4. Melatih model K-Means berdasarkan jumlah cluster terbaik
5. Visualisasi hasil cluster menggunakan scatter plot berbasis PCA
6. Interpretasi cluster dengan menganalisis karakter tiap kelompok pelanggan

Project ini memberikan gambaran dasar bagaimana teknik clustering dapat digunakan dalam dunia bisnis, khususnya pada bidang pemasaran, segmentasi pengguna, dan analisis perilaku konsumen.
