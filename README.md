# KMeans-Customer-Segmentation
Project ini merupakan latihan dari mata kuliah Machine Learning yang berfokus pada implementasi algoritma K-Means untuk melakukan segmentasi pelanggan. 

Tujuan dari project ini adalah memahami cara kerja teknik unsupervised learning dalam mengelompokkan pelanggan berdasarkan kesamaan perilaku dan karakteristik seperti usia, pendapatan, waktu penggunaan aplikasi, aktivitas web, jumlah pembelian, dan skor pengeluaran.

Pada project ini dilakukan beberapa tahapan utama, mulai dari preprocessing data, menentukan jumlah cluster terbaik menggunakan Elbow Method dan Silhouette Score, pelatihan model K-Means, visualisasi hasil cluster menggunakan PCA, hingga interpretasi karakter tiap segmen pelanggan. 

Hasil dari segmentasi ini dapat menjadi dasar untuk strategi pemasaran yang lebih terarah dan pemahaman yang lebih baik terkait perilaku konsumen.
